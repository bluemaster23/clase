# Clase 15-08-2023
## Instalación

### Clonar el repositorio:
```
    git clone https://github.com/bluemaster23/clase.git
    cd clase
```

### Instalación Manual

```
    npm install
```

## Tabla de contenido

### Característica
- Node js
- NPM

Run Local:
```
    npm run dev
```
Run Producción:
```
    npm run start
```
